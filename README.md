# vite-vue3-explore
explore the usage  of vite and build a vue3 project

## 快读搭建vite项目
```
$ yarn create vite-app <project-name>
$ cd <project-name>
$ yarn
$ yarn dev
```
## 即可初始化一个 vite 项目（默认应用模板为 vue3.x），生成的项目结构十分简洁：
```
|____node_modules // 依赖
|____App.vue // 应用入口
|____index.html // 页面入口
|____vite.config.js // 配置文件
|____package.json执行 yarn dev 即可启动应用
```
